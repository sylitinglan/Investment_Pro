# Investment_Pro
##前言
这是个quant新人的项目，用于安放自己做过的research的代码。我会尽量把这些代码包装的更实用一些，计划实现从数据调取，策略撰写，回测到交易接口的连接。

本人金融专业，还是希望自己以后的工作偏中台，做量化分析师可能会比较适合我。Investment_Pro可能更适合作为一个量化开发者锻炼的项目。但是通过开发脱离量化交易平台的框架来实现自己的个性化策略和增强quant原理的理解也是个好事。
